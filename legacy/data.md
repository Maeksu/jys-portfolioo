딥러닝 기반 분석을 통해 1~5분 단위의 초단기(스캘핑) 주식 매매 전략을 자동으로 생성·추천하는 웹 서비스입니다. 카카오톡 메시지 형태의 결과 알림과 직관적인 그래프 UI를 제공하여 실시간 매매 보조 도구로 활용됩니다.

- 전략별 기대 수익률, 성공률 분석
- 카카오톡 메시지 형태의 매매 전략 요약 제공

**사용 기술**

📌 PhishTrainer - 실전형 피싱 메일 시뮬레이션 및 인식 교육 플랫폼
설명
PhishTrainer는 실제 피싱 공격과 유사한 이메일 시뮬레이션을 통해 보안 인식을 강화할 수 있도록 돕는 웹 기반 플랫폼입니다. 사용자는 이메일을 입력하면, 피싱 메일이 자동으로 전송되며 클릭 시 가짜 로그인 페이지로 이동해 보안 경각심을 체험할 수 있습니다.

주요 기능

이메일 주소 입력 시 가짜 피싱 메일 자동 발송

훈련용 피싱 웹사이트 제공

행동 데이터 수집 (메일 열람 여부, 클릭 여부)

이메일 클릭 시 경고 팝업 제공

Firebase Hosting 및 Node.js 기반 이메일 서버 구성

사용 기술

React, TypeScript, Tailwind CSS

Node.js, Express, Nodemailer, dotenv

Firebase Hosting & Functions

SendGrid API (메일 발송 테스트 시 사용 가능)

HTML/CSS 기반 피싱 사이트

본인 역할

전체 시나리오 및 시스템 설계

이메일 발송 서버 구현 (Node.js + Nodemailer)

훈련용 피싱 웹페이지디자인 및 구현

Firebase Hosting을 활용한 정적 배포 환경 구성

React 기반 사용자 입력 UI 및 피드백 기능 구현

.env 환경변수 구성 및 API 연동 로직 처리
